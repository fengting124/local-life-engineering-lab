# LocalLife

`LocalLife` 是传统后端主项目。  
`LocalLife Copilot` 是配套 AI 应用项目。

当前仓库先完成三件事：

1. 固定开发环境。
2. 建立多模块工程骨架。
3. 沉淀完整项目文档。

## 当前目录

- `docs/`：项目文档
- `infra/`：本地开发基础设施
- `local-life-server/`：传统后端主项目
- `local-life-copilot/`：AI 应用项目

## 当前技术基线

- Java 17
- Maven 3.9.9
- Spring Boot 3.5.14
- Docker Desktop 29.x
- MySQL 8.4
- Redis 7
